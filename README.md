## ‚ú® Key Features

- **Modern UI:** Clean, dark-themed interface using `CustomTkinter`.
- **Live Frame Preview:** Real-time frame preview while dragging the sliders for precise cutting.
- **Video Trimming:** Easily select start and end points using the timeline sliders.
- **Audio Extraction:** Option to export the selected segment as an **MP3** file.
- **Mute Video:** Ability to remove audio from the video clip.
- **Speed Control:** Adjust video speed (e.g., 0.5x, 1.5x, 2.0x).
- **Real-time Progress Bar:** Accurate progress tracking during the export process.
